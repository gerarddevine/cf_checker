Announcement - March 2011

CF Checker updated to Version 2.0.3. This is a minor update. A summary of the changes follows:

  • Missing_value is no longer deprecated by the NUG
  • Fix bug where the checker was incorrectly complaining about a missing units attribute on some dimensionless variable
    • .
  • If the standard name modifier 'number_of_observations' is present then allow units to be omitted or units="1"
  • Fix bug with singleton aux-coord variable of type numpy.float64

Please email (r.s.hatcher@reading.ac.uk) if you encounter any problems.

Note: With UDUNITS-2 unit symbol comparisons are case-sensitive but unit name comparisons are not.

The previous version of the checker (vn2.0.2) is still available here.
